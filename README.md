# CPP Console Games

This repository contains simple console-based C++ projects created while learning programming fundamentals.

## Projects

### 1. Rock Paper Scissors Game

A console game where the player competes against the computer for multiple rounds.

#### Features

* Choose number of rounds
* Player vs computer gameplay
* Random computer choices
* Round winner detection
* Final game result summary
* Console color changes based on the winner

#### Concepts Used

* Enums
* Structs
* Functions
* Loops
* Random numbers
* Conditional logic
* Console input/output

---

### 2. Math Quiz Game

A console quiz application that generates random math questions based on difficulty level and operation type.

#### Features

* Choose number of questions
* Choose question level: Easy, Medium, Hard, or Mix
* Choose operation type: Addition, Subtraction, Multiplication, Division, or Mix
* Automatic answer checking
* Right and wrong answer counter
* Final pass/fail result

#### Concepts Used

* Enums
* Structs
* Arrays
* Functions
* Random numbers
* Loops
* User input validation
* Basic arithmetic operations

## Technologies Used

* C++
* Console Application

## Purpose

These projects were built to practice C++ fundamentals, problem solving, and basic console application logic.
